# 回避型依恋倾向自测 V2

这是一个纯前端单页 HTML 原型，包含“看懂 TA”和“了解我自己”双入口、两套独立题库、共享计分逻辑、四类依恋分型、分段答题、2 个呼吸缓冲页、计算加载动效和竖向阅读报告。

## 本地预览

直接打开 `index.html` 即可预览。也可以在当前目录启动静态服务后访问：

```bash
python3 -m http.server 4174 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:4174/
```

## 支付接口对接位置

V3 当前先做完整体验预览，没有展示价格或真实支付入口。后续可在报告生成前加入解锁页，支付成功并完成服务端订单校验后调用 `renderReport()`。同一订单 24 小时内可覆盖两种模式的解锁逻辑可以接在模式字段 `mode` 上。

## 合规说明

完整报告底部展示了 PRD 要求的重要说明。“看懂 TA”模式进入前有主观观察免责声明。页面不收集姓名、手机号、身份证号或地理位置，所有答题数据仅在浏览器当前页面内用于生成结果。
